El siguiente es folio es el contexto de mi proyecto de grado, en el cual debo hacer la creación de un aplicativo que automatice ciertos procesos del analisis forense integrando las siguientes herramientas: Volatility, Autopsy, TSK y WinPmem.

Debe ser un aplicativo .exe que cuando se ejecute muestre una interfaz que permita realizar el proceso de analisis forense integrando las herramientas antes mencionadas (es como una integracion de herramientas) y al final dar un reporte de los hallazgos del análisis.

Se puede tener el ingreso de informacion por medio de un disco duro externo con el volcado de memoria o sino ejecutar en la máquina que está sucediendo el ataque para realizar un analisis de memoria volatil y no volatil.

El lenguaje de programación es Python
